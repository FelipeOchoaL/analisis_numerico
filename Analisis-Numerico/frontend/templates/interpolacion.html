Métodos de Interpolación a Implementar

Esta sección incluirá los siguientes métodos de interpolación polinomial:

Método de Vandermonde

Utiliza una matriz de Vandermonde para encontrar el polinomio interpolante. Resuelve un sistema de ecuaciones lineales para obtener los coeficientes del polinomio.

Newton Interpolante

Utiliza diferencias divididas para construir el polinomio interpolante. Más eficiente que Vandermonde para agregar puntos adicionales.

Método de Lagrange

Construye el polinomio interpolante como una combinación lineal de polinomios base de Lagrange. Fácil de implementar y entender conceptualmente.

Spline Lineal

Conecta puntos con segmentos de línea recta. Simple pero puede tener discontinuidades en las derivadas.

Spline Cúbico

Utiliza polinomios de tercer grado entre cada par de puntos. Proporciona interpolación suave con derivadas continuas hasta segundo orden.

Funcionalidades Planificadas
  • Entrada de Datos: Interfaz intuitiva para ingresar hasta 8 pares de puntos (x, y)
  • Visualización: Gráficos interactivos que muestran los puntos datos y el polinomio interpolante
  • Polinomio Solución: Impresión del polinomio interpolante en formato legible
  • Informe Automático: Comparación de todos los métodos con análisis de errores
  • Análisis de Errores: Cálculo de diferentes tipos de errores para identificar el mejor método
  • Opción de Informe: El usuario puede elegir si ejecutar o no el informe comparativo
Cómo Ingresar los Datos

Cuando la interfaz esté lista, podrás ingresar los datos de las siguientes maneras:

  1. Tabla Interactiva: Una tabla donde puedes agregar hasta 8 pares de valores (x, y)
  2. Botones de Agregar/Eliminar: Controles para añadir o quitar puntos fácilmente
  3. Validación Automática: El sistema verificará que los datos sean válidos antes de procesarlos
  4. Ejemplo Pre-cargado: Datos de ejemplo para probar rápidamente los métodos
Nota: Los valores de x deben ser distintos entre sí para que la interpolación sea válida.
