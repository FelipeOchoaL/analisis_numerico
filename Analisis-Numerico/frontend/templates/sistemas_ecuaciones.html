Sistemas de Ecuaciones Lineales

Método de Eliminación de Gauss con Pivoteo
Método directo para resolución de sistemas lineales
Sistema de ecuaciones: Ax = b

Información del Método
Sin Pivoteo

Eliminación directa sin intercambio de filas. Puede fallar si hay pivotes cero o muy pequeños.

Pivoteo Parcial

Busca el mayor elemento en la columna actual e intercambia filas. Mejora la estabilidad numérica.

Pivoteo Total

Busca el mayor elemento en toda la submatriz e intercambia filas y columnas. Máxima estabilidad numérica.

Nota: Ingrese números decimales usando punto (.) como separador. El sistema debe ser cuadrado (mismo número de ecuaciones que incógnitas).
Método Iterativo de Jacobi
Converge si la matriz es diagonalmente dominante
Requisitos: Matriz cuadrada. Converge si ρ(T) < 1. Mejor para matrices grandes y dispersas con diagonal dominante.
Método Iterativo de Gauss-Seidel
Generalmente converge más rápido que Jacobi
Ventajas: Más rápido que Jacobi para matrices simétricas y definidas positivas. Usa valores actualizados inmediatamente.
Método SOR (Sobre-Relajación Sucesiva)
Acelera la convergencia con el parámetro ω adecuado
Parámetro ω: • ω = 1: Equivale a Gauss-Seidel • ω < 1: Sub-relajación • ω > 1: Sobre-relajación (acelera si se elige bien) • Rango válido: 0 < ω < 2
